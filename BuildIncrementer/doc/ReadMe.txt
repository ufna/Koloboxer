NightWind BuildIncrementer
v.1.0.0.1 2007-Mar-06
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

Программа для инкрементирования номера билда версии компилируемой программы.
Используется для файлов версий MSVC 6/7/8.
Инкрементирует последнее значение (билд) следующих полей:

"FILEVERSION"
"FileVersion"
"PRODUCTVERSION"
"ProductVersion"

Запускать как постбилд-эвент через ярлык.
Формат запуска:
"<путь к программе>" <путь к файлу ресурсов>
Путь к файлу ресурсов является относительным к папке, где лежит программа, либо необходимо установить другую рабочую папку программы (например, папку с файлом ресурсов).
